# 短期公告

有用户反馈在正式版更新后 Windows 通知报错，但复现数极少，大概率是精简版系统移除了不该移除的组件，或者单纯是 Windows 抽风了。

新版本已临时解决报错问题，但受到影响的用户仍然无法正常接收通知，我们无法提供除重装系统以外的修复建议，请避免使用未标明精简项及精简风险的“精简版”系统。
